# WE Video Picker
### Middleman software to connect wallpaper engine and KDE plugin "Smart Video Wallpaper Reborn"

## Quickstart
Run the following in a terminal

- to make the python file executable:
```
chmod +x we-video-picker.py
```

- the run the app: 
```
./we-video-picker.py
```
